# Attention-Only Noise

Scripts to replicate the Section 3.1 attention-only noise floor result.

Paper target:

```bash
python scripts/paper/run.py fig_attention_noise_floor --mode run-and-plot
```

Python modules:

```bash
python -m hebbian.expts.attention_noise.run_attention_only
python -m hebbian.expts.attention_noise.plot_pretty
python -m hebbian.expts.attention_noise.plot_sweep
```
